Workspace Management API -
A role-based workspace management backend system built with Node.js, Express, TypeScript, Prisma, and PostgreSQL.
It supports authentication, workspace management, projects, tasks, invitations, comments, notifications, and activity tracking.

Tech Stack -
Node.js, Express, TypeScript, Prisma, PostgreSQL, JWT, bcrypt, Zod, Multer
